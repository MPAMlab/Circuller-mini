# Roundoller-mini
A small &amp; round game controller that uses a 5.5' touchscreen and few buttons.

## Requirements

- 3D printed cases
- 5.5" MIPI to HDMI & USB touchscreen
- 8x Gateron low-profiled key switch
- Arduino Leonardo
- Demandable Computer w/ HDMI port
